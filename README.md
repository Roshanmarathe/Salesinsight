# 📊 Retail Sales Analytics Dashboard (SQL + Power BI)

## 🚀 Project Overview

This project delivers a complete **end-to-end data analytics solution** for a retail business using **SQL and Power BI**.  
The objective is to transform raw transactional data into **actionable business insights** through data modeling, KPI design, and interactive dashboards.

The project follows a real-world workflow used by data analysts:
- Data extraction using SQL  
- Data transformation and modeling  
- KPI creation using DAX  
- Dashboard design and storytelling  

---

## 🏢 About the Project Style (Inspired by Zoho Analytics)

This dashboard is designed with inspiration from modern BI tools like **Zoho Analytics**, focusing on:

- Clean and professional UI (dark theme)  
- KPI-driven decision-making  
- Interactive slicers for dynamic filtering  
- Multi-page analytical storytelling  

---

## 🗂️ Dataset Overview

The project uses a relational dataset with the following tables:

- **customers** → Customer details (demographics, location)  
- **orders** → Order-level information  
- **order_items** → Product-level transactions  
- **products** → Product details (category, pricing)  
- **shipping** → Delivery and logistics data  

---

## ⚙️ Project Workflow

### 🔹 1. Data Preparation (SQL)

- Data extracted and joined from multiple tables  
- Business metrics such as revenue and profit calculated  
- Aggregations and filtering applied  
- Clean dataset prepared for analysis  

---

### 🔹 2. Data Modeling (Power BI)

- Star schema model created  
- One-to-many relationships defined  
- Data types corrected  
- Duplicate records removed  
- Null values handled  

---

### 🔹 3. DAX Measures (KPI Creation)

Key business measures were created for:

- Revenue and Profit  
- Growth and Target tracking  
- Customer metrics (Retention, Acquisition)  
- Product performance  
- Shipping efficiency

## 🎛️ Slicers (Interactive Filters)

The dashboard includes dynamic filters for:

- Date range  
- Product category  
- State and city  
- Customer type (New / Returning)  
- Payment method  
- Gender  

These slicers allow users to perform **interactive and real-time analysis**.

---

## 📈 Business Insights

- Revenue shows a steady growth trend over time  
- A small group of products contributes most of the profit  
- Some products generate high sales but low profit margins  
- Returning customers drive the majority of revenue  
- Shipping delays negatively impact customer experience  
- Certain regions have higher logistics costs
